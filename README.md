⚖️ M.A | JUS IA EXPERIENCE — Auditoria Criminal de Elite

O M.A | JUS IA EXPERIENCE é um ecossistema de auditoria forense avançado, desenvolvido para o Direito Penal e Processual Penal. Esta aplicação desktop combina o poder de síntese do Gemini 2.5 Flash com o rigor determinístico de Escudos Anti-Alucinação via Regex.

🧠 Visão Estratégica

Diferente de IAs genéricas, este software foi construído para advogados criminalistas que exigem precisão absoluta. Ele resolve o problema das "alucinações" através de uma arquitetura de segurança em três camadas:

Busca Web Tática: Integração com DuckDuckGo para localizar jurisprudência real no Jusbrasil/STJ em tempo real.

Escudo Regex (A Porta de Aço): Filtra e omite numerações de processos para evitar que a IA invente dados inexistentes.

Análise de Volume em Lote: Processamento de múltiplos arquivos TXT, consolidando um parecer jurídico denso e dissertativo.

🛠️ Funcionalidades de Elite

Interface Tática (GUI): Abas organizadas para Direcionamento, Jurimetria, Análise Qualitativa e Nulidades.

Parecer Premium: Geração de documentos JSON estruturados que se transformam em pareceres de alta complexidade.

Dossiê Word Automatizado: Exportação direta para .docx com formatação profissional (margens de 3cm/2cm).

Checklist de Nulidades: Foco em buscas ilegais, quebra de cadeia de custódia e fragilidade probatória.

💻 Stack Técnica

Linguagem: Python 3.11+

UI: Tkinter (Interface Nativa customizada).

IA Engine: Google GenAI SDK (Gemini 2.5).

Processamento: Regex, JSON, Threading (para evitar travamento da UI).

Documentação: Python-Docx.

📂 Estrutura do Repositório

auditor_premium.py: Código-fonte principal da aplicação desktop.

requirements.txt: Dependências para instalação do ambiente.

.env.example: Modelo para configuração da chave de API.

🚀 Como Executar

Clone este repositório.

Instale as dependências: pip install -r requirements.txt

Crie um arquivo .env e adicione sua chave: GEMINI_API_KEY=sua_chave_aqui

Execute: python auditor_premium.py
